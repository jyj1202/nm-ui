# 简介

NM UI 是一个现代化的UI组件库，内容包括：
- 二次封装`element-plus`的form、table
- 原生`web component`组件
- 配套工具库
- 表单设计等相关插件

## 为什么选NM UI
NM UI设计的初衷是简单易用、灵活。
::: tip 提示

NM UI中二次封装`element-plus`部分受[Avue](https://avuejs.com/)启发。

:::

- 简单易用：
  - 不仅仅是使用简单，包括源码，也是非常简单。
  - ts支持，代码提示。
  - 配套工具库，节省时间。
- 灵活：
  - 按需引入，更好。
  - 原生web component，即使是不同框架，也能使用。
  - 也是因为源码简单易懂，更容易让初学者拓展。
